Primary Key

What is a primary key?

A primary key is the data that we use to identify a row in a table.
For example, in a table called student, you can use student_id as the primary key‚Ä¶or NIC‚Ä¶.You can‚Äôt use a name‚Ä¶because there can be multiple students with the same name.

Why do we use a primary key?

To prevent data from repeating, that is, to stop the same data from being inserted repeatedly, we use a primary key.
Another thing is that we use a primary key to distinguish and identify a record in the table.
There are two important points in what I just said. That is, the primary key must be unique. It cannot be repeated (the same primary key cannot be in two records).. Because it must be able to distinguish and identify.. The other is that the primary key cannot be empty (null)..
