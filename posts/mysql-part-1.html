MySQL Part 1

My purpose in writing this blog is to share how to create a database using MySQL. So if you are someone who wants to get into the programming side and learn about MySQL, this article will be very important for you.

Before learning, let's see what is the importance of MySql...

After reading this report, you will understand how important MySQL is. Currently, the most used database by learners is MySQL. MySQL is a top data management system used not only by learners but also by professional developers.

What is a database?

If a data set is stored in one place using some medium, we can call it a database. In other words, it is an organized data set. The data we store is permanently stored on the hard disk.

Why use a database?

As mentioned earlier, a database is used to store data. And after storing this data, we can retrieve it at any time. As an example, think of it this way. Almost all of us have a Facebook account. When we log in to the account, we can only log in if we enter the username and password correctly. What happens is that our username and password are stored in a database and when we log in again, the username and password we enter are checked to see if the previously stored username and password match. Not only can we insert and retrieve data, but we can also do things like update and delete that data.

How to store data in a database

Data in a database is stored as a table. Therefore, it is easy for us to use that data. When data is stored, the data is stored as tuples in that relevant table. Similarly, each table has a unique name. Let's see how that name comes about.
